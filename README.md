# 🎫 SmartTicket Pro: Gestión Inteligente de Tareas y Solicitudes

### **Transformando la eficiencia operativa con Google Apps Script**

**SmartTicket Pro** es una solución empresarial de vanguardia diseñada para centralizar, transparentar y acelerar la gestión de requerimientos internos. Olvídate de los correos perdidos y las solicitudes por chat; nuestra plataforma ofrece un ecosistema fluido donde la trazabilidad y la rendición de cuentas son la norma.

---

## 🚀 La Solución: ¿Qué aporta a tu empresa?

En un entorno corporativo dinámico, la comunicación suele ser el cuello de botella. **SmartTicket Pro** llega para resolver este problema mediante:

1.  **Centralización Total:** Un único punto de entrada para errores, mejoras, consultas y proyectos.
2.  **Trazabilidad Extrema:** Sepa exactamente quién solicitó qué, a quién, y en qué estado se encuentra en tiempo real.
3.  **Seguridad y Accesibilidad:** Acceso personalizado mediante DNI vinculado a una base de colaboradores activos.
4.  **Respaldo Digital:** Soporte nativo para evidencia visual (imágenes) y documentación técnica (PDF) almacenada de forma segura en Google Drive.
5.  **Data-Driven Decisions:** Toda la información se vuelca automáticamente en un Google Sheet, permitiendo generar reportes y KPIs de rendimiento de forma instantánea.

---

## ✨ Características Principales

*   **🔐 Autenticación Smart:** Login seguro basado en DNI, validando automáticamente el área y el estado del colaborador.
*   **📊 Tableros de Control Dual:**
    *   **Vista Requerida:** Seguimiento en tiempo real de los tickets enviados.
    *   **Vista Operativa:** Gestión activa de tickets recibidos, permitiendo actualizar estados, fechas estimadas y observaciones finales.
*   **📑 Gestión Documental:** Adjunta fotos y PDFs para una comprensión clara del requerimiento, eliminando ambigüedades.
*   **📈 Dashboard de Indicadores:** Visualización rápida de carga de trabajo (Pendientes, En Proceso, Finalizados) para cada usuario.
*   **☁️ Tecnología Cloud:** Basado íntegramente en Google Workspace, sin costos de infraestructura adicionales y con disponibilidad 24/7.

---

## 🛠️ Especificaciones Técnicas

*   **Backend:** Google Apps Script (Javascript V8 Runtime).
*   **Frontend:** HTML5, CSS3 (Modern Dark Theme) y Javascript SPA.
*   **Base de Datos:** Google Sheets (Estructura de 12 columnas optimizada).
*   **Almacenamiento:** Google Drive API para anexos.
*   **Diseño:** Interfaz Premium inspirada en principios de Glassmorphism y Dark Mode para reducir la fatiga visual.
